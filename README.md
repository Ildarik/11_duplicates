# Anti-Duplicator

Find duplicate files (name & size) in your computer. 

# How to use
```bash
python3 duplicates.py /home/ildar/
```
Example output:
```bash
DUPLICATES:
/home/ildar/Downloads/gutenprint-5.2.11/m4/ltsugar.m4
/home/ildar/Downloads/curl-7.47.1/m4/ltsugar.m4
```
